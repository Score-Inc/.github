from : \
![after-pkg][after-pkg]\
To : \
![before-pkg][before-pkg]

6. > Click on Yuuki Server
7. > Enter username and password but easy to remember\
![Login_Page](/YuukiPS/src/img/Android/Clone%20Version/Login_Page.png)
8. Enjoy
